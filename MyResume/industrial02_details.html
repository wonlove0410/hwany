Project Overview

이 의자는 철로 제작한 튼튼한 틀을 바탕으로, 좌판과 등받이 부분을 직접 끈으로 엮어 제작한 독창적인 디자인입니다. 끈을 엮는 방식은 자연스러운 탄성을 제공하여 사용자가 장시간 앉아도 편안함을 유지할 수 있게 해주며, 인체 공학적 설계로 인해 체중 분배가 균형 있게 이루어집니다. 좌판의 높이와 등받이 넓이는 세심한 계산을 통해 최적의 앉는 자세를 유지할 수 있도록 디자인되어, 장시간 사용에도 피로감을 최소화합니다. 각 부분은 정밀한 제작과 고유한 수공예적 요소가 결합되어, 단순한 의자 이상의 편안함과 기능성을 갖춘 예술적 작품으로 완성되었습니다.
